# Security policy

FlowPlane is pre-GA. Do not include secrets, customer data, production payloads, internal URLs, or exploit details in a public issue.

To report a vulnerability, open a minimal GitHub security advisory for this repository if private reporting is enabled. Otherwise contact the repository owner through the organization profile and request a private disclosure channel. Include the affected component and revision, reproduction conditions, impact, and a safe proof of concept.

The maintainers will acknowledge receipt, triage severity and scope, coordinate remediation, and publish disclosure details only after a fix or mitigation is available. No response-time SLA is promised for this pre-GA project.

The evidence in this repository demonstrates selected implemented controls and test outcomes. It is not a penetration test, compliance attestation, or security certification.
